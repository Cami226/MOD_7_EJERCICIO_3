# MOD_7_EJERCICIO_3


M7_AE3_ABP-Ejercicio individual[Actividad Evaluada]



Ejercicio individual
Objetivo

El estudiante creará y publicará un portafolio técnico utilizando GitHub y Behance, además de explorar opciones de hosting y presentación en video para mostrar su trabajo de manera efectiva.

Instrucciones

1. Creación de un Portafolio en GitHub

Configura tu cuenta de GitHub (si no tienes una).

Crea un repositorio público llamado "Portafolio".

Diseña un README.md atractivo siguiendo estas buenas prácticas:

Breve presentación personal.

Lista de proyectos destacados con enlaces.

Tecnologías que dominas.

Imagen o banner llamativo.

Sube al menos un proyecto con estructura clara, documentación y un archivo LICENSE.

2. Creación de un Portafolio en Behance

Regístrate en Behance (si no tienes cuenta).

Crea una presentación de un proyecto UX/UI que hayas realizado.

Aplica buenas prácticas en la publicación:

Descripción del problema y la solución.

Prototipos o wireframes visuales.

Proceso de diseño y herramientas utilizadas.

Reflexión sobre el aprendizaje en el proyecto.

3. Hosting y Publicación de Proyectos

Explora opciones gratuitas de hosting: GitHub Pages, Netlify, Vercel o Render.

Aloja un proyecto en un servicio cloud siguiendo estos pasos:

Si es un sitio estático, usa GitHub Pages o Netlify.

Si es un proyecto con backend Python, usa Render o Railway.

Si es un proyecto UX/UI, usa Notion o Figma como landing page.

Comparte el enlace del hosting en tu portafolio de GitHub.

4. Creación de un Video Técnico en YouTube

Utiliza herramientas gratuitas como OBS Studio, Loom o Screencast-O-Matic para grabar pantalla.

Graba un tutorial técnico explicando tu proyecto en 3-5 minutos.

Presentación del proyecto.

Explicación de su funcionalidad.

Tecnologías utilizadas.

Código o diseño clave.

Sube el video a YouTube y agrégalo en tu GitHub y Behance.
